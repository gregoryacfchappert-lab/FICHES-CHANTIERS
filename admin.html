⚙️ Configuration GitHub (à faire une seule fois)
Pourquoi ? C'est ce qui permet au bouton « Publier » de mettre la fiche en ligne tout seul (voir LISEZMOI.txt pour la mise en place complète du site).
  1. Compte : votre nom d'utilisateur GitHub (ex : acfchappert).
  2. Dépôt : le nom du dépôt où est le site (ex : fiches-chantier).
  3. Jeton : sur github.com → photo de profil → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Repository access : Only select repositories → votre dépôt. Permissions → Contents : Read and write. Copiez le jeton ci-dessous.
  4. Cliquez « Tester la connexion ». Ces réglages restent enregistrés dans votre navigateur uniquement (jamais publiés sur le site).
